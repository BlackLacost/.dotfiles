### BetterTags: Advanced Configuration

These advanced settings do not sync and require a restart to apply.

- **enableHierarchicalTags** [true/false]: Whether or not to enable Hierarchical Tags features. Default: `true`
- **enableHierarchicalCompleter** [true/false]: Whether or not to enable Hierarchical Completer feaures. Default: `true`
- **hierarchicalCompleterArbitrarySubstrings** [true/false]: Allow arbitrary subtag matches instead of limiting autosuggestions to tag hierarchy. Default: `false`
- **hierarchicalCompleterHighlightWorkaround** [true/false]: Use a workaround for displaying substring highlights (required on some Linux systems). Disable this if you would like your autocompleter to look a bit nicer. Default: `true`
- **hierarchicalTagsSeparator** [text]: Separator used by Hierarchical Tags. Default: `::`
- **hotkeyFindReplace** [text]: Hotkey that calls 'Find and Replace Tags' dialog. Default: `Ctrl+Alt+Shift+T`
- **hotkeyRefreshTags** [text]: Hotkey that refreshes tags in the browser. Default: `Ctrl+Shift+F5`
- **hotkeyDeleteTag** [text]: Hotkey that calls the 'Delete' function when the sidebar is focused. Default: `Delete`
- **hotkeyRenameTag** [text]: Hotkey that calls the 'Rename' function when the sidebar is focused. Default: `F2`
